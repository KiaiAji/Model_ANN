# Model_ANN
Proyek ini berisi implementasi **Artificial Neural Network (ANN)** menggunakan dataset **Iris**.  
Model dibuat dengan **TensorFlow/Keras** untuk mengklasifikasikan spesies bunga Iris berdasarkan fitur morfologisnya.
